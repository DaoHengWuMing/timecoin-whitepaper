# **时币：时间本位去中心化货币白皮书**

# TimeCoin Whitepaper

> Fair money generated by online time, not by capital.

---

## 📖 Abstract

本白皮书提出首个纯时间锚定型去中心化货币体系。
任何联网设备只需保持在线即可赚取时币，无需投入资本、无技术门槛、也无需身份核验。

This whitepaper proposes the first purely time-anchored decentralized currency system.  
Every device with an Internet connection can earn time-coins by simply being online — no capital, no technical barriers, no identity verification.  

Key mechanisms:
- **One device, one identity, one wallet** – based on PUF (Physically Unclonable Function)
- **Time-based issuance** – verifiable delay functions (VDF) make time unforgeable
- **Fully transparent** – every coin carries its complete transaction history
- **Anti-capital extraction** – zero marginal cost for ordinary users makes price manipulation mathematically impossible

---

## 📜 Core Rules

1. **One device, one vote, one wallet** – Every physical device has a unique identity bound to its PUF chip.  
2. **Time-anchored & fully transparent** – Each time-coin is minted by verifiable online time, with a complete, immutable transaction chain.  
3. **Real-time governance by all nodes** – All parameters are adjusted by continuous weighted voting of active nodes.
